### 1.1 纳指100是什么

纳斯达克100指数 = 按市值加权选取纳斯达克交易所中最大的100家非金融公司。

### 1.3 优胜劣汰机制（最关键的制度保障）

**纳指100不是固定的一篮子股票。它有自动的优胜劣汰机制：**

```
调整规则：
  - 每季度审核一次成分股
  - 按市值排名，大的进来，小的出去
  - 权重按市值动态调整——市值越大权重越高

这意味着什么？
  - 某家公司不行了（如Nokia、BlackBerry、Yahoo）
    → 市值缩水 → 自动被踢出指数
    → 持仓自动从弱者转移到强者身上

  - 某家新公司崛起（如英伟达、特斯拉）
    → 市值暴涨 → 自动被纳入指数
    → 持仓自动追踪当前市场上最好的公司

关键结论：
  → 指数始终持有当下全球最强的100家科技公司
  → 省去了主观判断哪家公司好的难题，指数机制替投资者完成了选股
  → 被动管理的优势在于：规则透明、制度驱动，不依赖个人判断
```

**历史上被淘汰的公司举例：**

```
被淘汰              原因                    替代者
──────────────────────────────────────────────────────
Nokia              智能手机时代掉队          苹果/三星
BlackBerry         触屏革命失败              苹果
Yahoo              搜索/广告被谷歌超越       谷歌
AOL                门户网站时代终结          Meta/谷歌
Sun Microsystems   服务器被云计算替代        亚马逊AWS
Intel              CPU被AMD/英伟达超越       英伟达/AMD
```

**总结**：纳指100本质上是一台"自动换血"的机器，持续淘汰落后者、吸纳领先者。赢家会自动进入指数成分，无需个人判断。

## 第二章：定投为什么不需要择时

### 2.1 定投 vs 一次性买入

```
同样的100万（以下以人民币为例），两种投入方式，10年后的差距：

方式一：一次性买入100万
  假设10年间纳指波动上涨，最终涨幅30%
  → 最终资产 = 100万 × 1.30 = 130万
  → 收益率 = 30%
  → 风险：如果买入后先跌40%，需要涨67%才能回本
  → 心理压力：100万变成60万，对大多数人来说很难淡定

方式二：每月定投8333元（100万/120个月）
  第1年：高位买入，份额少
  第2年：市场回调，低位买入更多份额
  第3年：继续低位积累
  第4年：市场开始反弹，低位筹码开始盈利
  ...
  第10年：最终涨幅同样是30%
  → 但你在低位买到了更多份额
  → 平均成本 < 一次性买入的价格
  → 收益率 > 30%

数学本质：
  一次性买入的收益 = 最终价 / 买入价
  定投的收益       = 最终价 / 加权平均成本

  只要中间有过下跌，加权平均成本 < 起始价格
  → 定投收益率 > 一次性买入收益率

  而纳指100在任何一个10年窗口内都有多次回调
  → 定投必然比一次性买入成本低
```

### 2.2 定投的数学原理：成本平滑效应（微笑曲线）

```
假设纳指净值按月变动如下，每月投入10000元：

月份    净值    买入份额    累计投入    累计份额
──────────────────────────────────────────────
1月     1.00    10000份     10000      10000
2月     0.80    12500份     20000      22500    ← 下跌买更多份额
3月     0.70    14286份     30000      36786    ← 继续跌，买更多
4月     0.90    11111份     40000      47897
5月     1.00    10000份     50000      57897
6月     1.10     9091份     60000      66988
7月     1.20     8333份     70000      75321

平均买入成本 = 70000 / 75321 = 0.929元
当前净值 = 1.20元
收益率 = (1.20 - 0.929) / 0.929 = +29.2%

而净值从1.00到1.20只涨了20%
→ 定投因为低位多买了份额，收益 > 净值涨幅
→ 这个现象被称为"微笑曲线"
```

### 2.3 定投标的选择原理：为什么定投纳指而不是别的

```
不是所有资产都适合定投。定投的效果取决于资产的走势形态：

形态一：缓慢上涨（如银行存款、短债基金）
  净值走势：→→→→→→→→→→
  定投效果：每次买入价都比之前高，成本被不断抬升
  结论：不如一次性买入。定投反而让你少赚了早期的涨幅

形态二：持续下跌（如退市的股票、衰退行业）
  净值走势：↓↓↓↓↓↓↓↓↓↓
  定投效果：越买越亏，投入越多亏损越大
  结论：定投也救不了。买入一个长期向下的资产，越定投越惨

形态三：震荡向上（如纳指100、标普500）
  净值走势：↗↘↗↗↘↗↗↗↘↗
  定投效果：
    ↘ 回调时买入更多份额（打折买入）
    ↗ 上涨时份额跟着增值（盈利放大）
    长期向上确保最终价格 > 平均成本
  结论：适合定投。回调反而带来了更多低价买入的机会

为什么纳指100属于"震荡向上"：
  - 底层公司（苹果/微软/英伟达）持续创造利润
  - 优胜劣汰机制确保指数始终代表最强企业
  - 科技产业趋势持续推动（移动互联→云计算→AI）
  - 30年年化约12%，过程中有多次20-40%的回调
  → 回调提供低价买入机会，趋势向上确保最终盈利
  → 纳指定投 = 自动在每次"恐慌"时抄底

总结：
  定投的本质是"用时间换空间"
  → 只有"长期向上但中间有波动"的资产，定投才能发挥最大效果
  → 纳指100历史上恰好符合这个特征
```

### 2.4 择时 vs 不择时的回测对比

```
以下数据建议通过第八章的回测代码自行验证，此处为近似结果：

回测条件：纳指100，2013-2026，每月定投

策略                          年化收益    说明
──────────────────────────────────────────────────────────
固定金额无脑定投               ~12%     基准
跌30%加倍投入                 ~10-11%  踏空上涨月份
涨50%减半投入                 ~10%     踏空更多
RSI调仓（低买高卖）           ~10-11%  频繁调仓摩擦
等跌10%再买（择时）           ~11%     踏空+投入减少

→ 无脑定投年化约12%，在各策略中表现最优
→ 原因：纳指70%以上的月份在上涨
→ "等跌再买"或"涨了减仓"等策略，在回测中都因踏空上涨月份而表现不如固定定投
→ 在一个长期上涨的资产上，减少买入意味着错过上涨

注：以上回测未扣除基金管理费、托管费、申购赎回费、汇兑损失，
实际收益会略低于测算值。详见第九章风险提示。
```

### 2.5 为什么"高点买入"在长期定投中不成立

```
很多人怕"现在价格太高，等跌了再买"

事实：纳指100在2013-2026年间，
  - 创历史新高的月份占比 > 35%
  - 如果你在任何一个"历史高点"开始定投，
    坚持3年以上，历史上没有一个场景是亏损的

  注意：这是历史回测结果，不代表未来一定如此。

原因：
  定投不是一次性买入，而是持续分散买入
  你买入的均价 = 整个定投期间的加权平均价
  只要资产长期向上，均价 < 最终价格

核心逻辑：
  定投的本质是"时间分散"：
  → 资金不是在一个时间点全仓买入，而是在N个时间点分别买入
  → 每个时间点的买入价不同，有高有低
  → 最终你的成本是这N个价格的加权平均
  → 只要资产最终价格高于加权平均成本，你就盈利
  → 回调越多、越深，你在低位买入的份额占比越大，成本越低
```

## 第三章：为什么不需要在高点卖出

### 3.1 "泡沫高点"不断被刷新的事实

```
纳指100历史"泡沫高点"：

年份    当时的"高点"    后来怎样
────────────────────────────────────────────
2000    4800点          互联网泡沫破裂→跌到1100
                        2000年高点持有到2026年仍盈利(含分红再投资)
2007    2800点          金融危机→跌到1200
                        2007年高点到2026年涨了数倍
2015    4700点          "科技泡沫2.0"→继续涨
                        到2026年涨了约4倍
2021   16700点          "疫情泡沫"→2022跌到11000(-34%)
                        到2026年创新高

规律（历史经验，非预测）：纳指的每一个"泡沫高点"在5-10年后回头看都是低谷

重要提醒：这不意味着未来一定如此。详见第九章风险提示。
```

### 3.2 卖出的机会成本

```
研究数据（标普500，1928-2025，学界广泛引用）：
  错过最好的10个交易日 → 年化从约10%降到约6%
  错过最好的20个交易日 → 年化从约10%降到约3%
  错过最好的30个交易日 → 年化从约10%降到约1%

最好的交易日往往出现在最恐慌的时候：
  2008年10月28日:  +11.6%  (金融危机最恐慌时)
  2020年3月24日:   +9.4%   (疫情熔断后反弹)

→ "逃顶"看似在保护利润
→ 但实际操作中，大概率会错过最好的反弹日
→ 卖出后还需要判断什么时候买回来
→ 而历史上最好的买入时机，往往是最不敢买的时候
```

### 3.3 持有 vs 逃顶的权衡

```
假设你有完美的预判能力，每次纳指跌20%之前精确卖出，跌20%后精确买回：

  完美逃顶策略:    年化约13-14%
  长期持有:         年化约14-15%

→ 即使完美逃顶，收益也未必超过长期持有
→ 因为买卖的摩擦成本+踏空上涨+税收

那如果不完美呢？
  普通人逃顶（卖早了/买晚了）:  年化约8-10%
  → 远不如长期持有的14-15%

更合理的理解：
  不是说"永远不应该卖出"
  而是"判断什么时候卖"这件事本身极难做对
  对于没有专业分析能力、没有时间盯盘的普通人，
  长期持有是期望收益最高的策略
```

## 第四章：为什么这套策略特别适合程序员

### 4.1 程序员的收入特征

```
程序员职业的三个核心特征：

特征一：35岁前现金流极高
  应届生(22岁):     年薪15-25万
  3年经验(25岁):    年薪30-50万
  5年经验(27岁):    年薪50-80万
  高级/架构(30岁):   年薪80-150万
  35岁以后:         不确定性陡增

  → 25-35岁是黄金积累期，现金流远超同龄人
  → 但这个窗口期只有10年，值得好好利用

特征二：35岁后的不确定性
  行业裁员潮（2022-2024已发生多轮）
  技术迭代快，学习能力下降
  体力下降，加班能力下降
  公司偏好更年轻的工程师

  → 收入下降的概率不容忽视
  → 建议在收入高的时候为未来可能的变化做准备

特征三：时间非常值钱
  时薪估算（按2000工时/年）：
    年薪30万:  时薪150元
    年薪50万:  时薪250元
    年薪80万:  时薪400元
    年薪120万: 时薪600元

  → 每小时值150-600元
  → 花1小时研究个股，相当于投入了150-600元的隐性成本
```

### 4.2 研究个股 vs 定投的时间成本对比

```
时间成本分析：

假设你花时间研究个股/择时：
  每天研究1小时 × 250个交易日 = 250小时/年
  按时薪250元计算 = 隐性成本 62,500元/年

结果如何？
  业余散户炒股胜率: 约30%（70%的人跑输大盘）
  即使用心研究，普通人的超额收益: 年化 -2% ~ +2%
  (负数是因为频繁交易的手续费+印花税+追涨杀跌)

对比定投：
  每月操作时间: 1分钟 × 12 = 12分钟/年
  时间成本: 约50元/年

差额：
  研究个股的隐性成本: 62,500元/年
  定投的时间成本:        50元/年

→ 把省下的250小时投入工作：
    提升技术 → 升职加薪
    做副业/开源 → 简历增值
    学新技能 → 职业护城河

→ 对于绝大多数程序员，把时间投入职业发展的期望收益
  远大于把同等时间用于研究个股
```

### 4.3 定投是程序员的"自动化部署"

```
程序员最擅长的就是自动化。定投本质上就是：

  手动操作（每次都要决策）:
    看盘 → 分析 → 犹豫 → 决定 → 买入/卖出 → 后悔 → 循环
    ↑ 这跟手动部署服务器一样低效且容易出错

  定投（自动化流水线）:
    设置 → 每月自动执行 → 持续运行
    ↑ 类似 CI/CD 的投资版

  把投资变成一个 cron job：
    - 每月触发一次
    - 执行固定逻辑（买入固定金额）
    - 无需人工干预
    - 不受情绪影响
    - 结果可回测验证

  → 程序员比任何人都更应该理解：
    系统化 > 手动操作
    自动化 > 人为判断
    规则 > 情绪
```

### 4.4 25-35岁：窗口期与策略的匹配

```
这套策略的时间线和程序员的职业周期较为契合：

      → 收入快速增长 → 每年加5万
      → 不需要花时间研究投资 → 全力投入职业发展

      → 即使遇到裁员/收入下降
      → 多年积累的资产持续以复利增长
      → 财务压力大幅降低

40岁+: 资产持续复利增长
      → 可以选择降薪做喜欢的工作/创业/自由职业
      → 职业选择更加从容

关键洞察：
  → 程序员的高收入窗口期(25-35) 恰好是定投的最佳积累期
  → 程序员的高时薪 恰好是"不用花时间研究投资"的最大理由
  → 程序员的自动化思维 恰好是"规则化定投"的最佳执行者
```

### 6.5 复利的本质

```
复利公式：FV = PV × (1 + r)^n

关键变量不是本金(PV)，而是时间(n)和收益率(r)

为什么时间比本金重要？
  以版本A（每年固定25万，累计投入250万）为例，年化12%：
    10年后(34岁):    491万    翻2.0倍
    20年后(44岁):  1,526万    翻6.1倍
    30年后(54岁):  4,740万   翻19.0倍
    35年后(59岁):  8,353万   翻33.4倍

  → 前10年翻了2倍
  → 后25年翻了17倍
  → 越往后增长越快（指数函数的特性）

  → 所以：开始得早 比 投入得多 更重要
  → 25岁开始投入25万/年 可能比 35岁开始投入50万/年 最终结果更好
```

### 6.6 4%法则：资产到了多少可以"退休"

```
4%法则（Trinity Study，三一大学1998年研究）：

  每年从总资产中提取4%用于生活开销
  在股票+债券组合中，30年内资产不会耗尽的概率 > 95%
  原研究基于美国历史数据，适用于长期持有股票类资产

为什么是4%？

  纳指100长期年化收益: ~12%（历史近似值）
  通胀平均: ~3%
  实际增长(扣除通胀): ~9%

  每年拿走4% → 剩余增长 ~8% → 资产还在持续增长
  → 每年花的钱理论上永远花不完，资产反而越来越大

  数学验证（年化12%，每年年初提取4%）：
    假设资产1000万：

    年初资产    提取4%    剩余    年末(×1.12)
    ──────────────────────────────────────────
    1000万     40万     960万    1075万    → 净增长75万
    1075万     43万    1032万    1156万    → 净增长81万
    1156万     46万    1110万    1243万    → 净增长87万

    → 每年花40万+，资产反而从1000万涨到1243万
    → 只要年化收益 > 提取比例，资产就永远在增长
```

```
不同资产规模下的年可支配开销（4%法则）：

  总资产        每年可花(4%)    每月可花
  ──────────────────────────────────────────
  500万         20万           1.67万
  800万         32万           2.67万
  1000万        40万           3.33万
  1500万        60万           5.00万
  2000万        80万           6.67万
  5000万       200万          16.67万

→ 1000万资产 = 每年40万被动收入 = 每月3.3万
→ 不需要工作，不需要动用本金，靠资产增长就够了
```

```
回到复利模型，看看每个阶段的"退休可行性"：
（以下以版本A为例，数据由 compound_calculator.py --increase 0 生成）

39       866万       34.6万      二线城市基本生活

49     2,690万      107.6万      绰绰有余
59     8,353万      334.1万      完全自由

重要前提：
  → 以上基于年化12%的假设，实际收益可能不同
  → 4%法则基于美国历史数据，存在失效的可能性
  → 如果某几年年化低于4%（如2022年纳指跌37%），当年需要降低提取比例
```

#### 回测示例代码

```python
"""
最简纳指定投回测：验证本报告中的数据
依赖: pip install akshare pandas
"""
import akshare as ak
import pandas as pd

def backtest_dca(fund_code="161130", start="20130101", end="20261231", monthly=1000):
    """模拟每月固定金额定投"""
    df = ak.fund_open_fund_info_em(symbol=fund_code, indicator="单位净值走势")
    df.columns = ["date", "nav"]
    df["date"] = pd.to_datetime(df["date"])
    df = df[(df["date"] >= start) & (df["date"] <= end)]

    # 取每月第一个交易日
    monthly_df = df.groupby(df["date"].dt.to_period("M")).first().reset_index(drop=True)

    total_shares = 0.0
    total_invested = 0.0
    records = []

    for _, row in monthly_df.iterrows():
        shares = monthly / row["nav"]
        total_shares += shares
        total_invested += monthly
        current_value = total_shares * row["nav"]
        records.append({
            "date": row["date"],
            "nav": row["nav"],
            "monthly_invest": monthly,
            "total_invested": total_invested,
            "total_shares": round(total_shares, 2),
            "market_value": round(current_value, 2),
            "profit_pct": round((current_value / total_invested - 1) * 100, 2),
        })

    result = pd.DataFrame(records)
    final = result.iloc[-1]
    years = len(result) / 12
    cagr = (final["market_value"] / final["total_invested"]) ** (1 / years) - 1

    print(f"回测区间: {start} ~ {end}")
    print(f"月投入: {monthly}元")
    print(f"定投月数: {len(result)}个月")
    print(f"累计投入: {final['total_invested']:,.0f}元")
    print(f"最终资产: {final['market_value']:,.0f}元")
    print(f"总收益率: {final['profit_pct']:.1f}%")
    print(f"年化收益: {cagr*100:.1f}%")

    # 找最大回撤期间
    result["drawdown"] = (result["market_value"] - result["total_invested"]) / result["total_invested"]
    worst = result.loc[result["drawdown"].idxmin()]
    print(f"最大浮亏: {worst['drawdown']*100:.1f}% ({worst['date'].strftime('%Y-%m')})")

    return result

# 运行: 回测2013-2026每月定投1000元
backtest_dca(fund_code="161130", start="20130101", end="20261231", monthly=1000)
```

## 第九章：风险提示

> 任何投资策略都有风险。以下是你需要了解的真实风险。

### 9.1 流动性风险（最重要的风险）

```
定投纳指的10年间，资金被"锁"在基金里：
  → 纳指是长期投资，短期卖出可能亏损
  → 如果中途遇到大开销（买房、结婚、看病）需要赎回
  → 恰好碰上纳指在回撤期 → 实际亏损可能很大

建议：维持应急金
  → 至少3-6个月生活费放余额宝/货基，随时可取
  → 定投的钱 = 未来5-10年不需要用的钱
  → 不建议把全部现金都投入定投

  如果没有应急金就遇到大开销：
    例：定投3年，累计投入90万，恰逢纳指跌30%
    → 被迫赎回 → 实际到手约75万 → 亏损15万
    → 这更多是资金规划层面的问题，而非策略本身的问题
```

### 9.2 操作变形风险

```
定投策略的核心难点不在于"懂不懂"，而在于"能不能坚持"：

恐慌停投：
  纳指跌30%的时候，新闻铺天盖地都是"崩盘""泡沫破裂""经济衰退"
  → 很多人在这时候害怕了，暂停定投甚至赎回
  → 但回测数据显示，这恰恰是最好的定投时机（低位买到更多份额）
  → 停投的结果：错过了后面反弹带来的收益

杀跌跑路：
  纳指跌40%的时候，账面浮亏可能达到20-30%
  → 心理压力极大，很多人选择"先跑再说，等稳了再买回来"
  → 但历史上最好的反弹日往往出现在最恐慌的时候（见第三章 3.2）
  → 跑了之后再想买回来，大概率已经错过了最低点

为什么难以坚持：
  → 人天生对亏损的痛苦感是同等盈利快乐感的2倍（损失厌恶）
  → 看到账户亏钱，理性告诉你要坚持，但本能告诉你要跑
  → 身边人的恐慌情绪会互相传染

应对方式：
  → 提前做好心理预期：纳指跌30-40%是正常现象，历史上发生过多次
  → 设置定投日历提醒，到时间就执行，不看盘
  → 回顾第五章的极端行情数据：每次大跌后都回来了
  → 和信任的朋友互相监督，避免独处时冲动操作
```

### 9.3 消费克制风险

```
高比例定投意味着每月大量现金被锁定，对消费习惯是一个挑战：

现实压力：
  → 年薪30万的人，每年投入25万定投 ≈ 月薪的83%
  → 剩下的钱要覆盖房租/房贷、吃饭、交通、社交、学习
  → 这意味着在定投期间，生活方式需要相对克制

常见陷阱：
  → 看到同事换新手机/新车/旅游，自己却在"攒钱买基金"
  → 社交压力：朋友聚餐、同事请客，长期回避会疏远关系
  → 紧急情况：家里突然需要用钱，但定投资金不想动

应对方式：
  → 量力而行：如果25万/年压力太大，可以从15万甚至10万开始
  → 不建议为了高定投而牺牲基本生活质量和健康
  → 用 compound_calculator.py 分别测算不同投入金额的结果，
    即使每年只投10万，复利效果依然可观
  → 随着收入增长逐步增加投入，而不是一开始就拉满

关键原则：
  → 定投是为了让未来的生活更好，而不是让现在的生活更差
  → 找到投入和生活质量之间的平衡点，才能长期坚持
```

### 9.4 行业集中风险

```
纳指100高度集中于科技行业：

  → 如果科技行业整体进入长期衰退（历史上尚未发生过，但不代表不可能）
  → 纳指可能长时间不涨甚至下跌
  → 互联网泡沫后纳指用了约15年才创新高（2000-2015）

缓解方式：
  → 不敢承受集中风险的 → 标普500（覆盖所有行业）
  → 折中方案 → 纳指50% + 标普50%
  → 标普500在互联网泡沫期间只跌了49%（纳指跌78%），恢复更快
```

### 9.5 汇率风险

```
国内投资者通过QDII基金买纳指，存在汇率波动：

  → 基金用人民币买美元资产
  → 如果人民币大幅升值 → 美元资产换回人民币变少 → 收益被汇率吃掉
  → 如果人民币贬值 → 反而多赚

  历史上汇率波动每年约2-5%，长期来看方向不确定
  这是国内投资者买海外资产绕不开的成本
```

### 9.6 QDII基金限制

```
国内买纳指只能通过QDII基金，存在以下现实问题：

  1. 限购：纳指QDII基金经常限购（额度紧张）
     → 有时每天只能买1000元，甚至完全暂停申购
     → 这是常态，需要有心理准备和替代方案

  2. 费率：QDII基金费率高于普通基金
     → 管理费约1.0-1.2%/年（普通基金约0.5%）
     → 托管费约0.2-0.25%/年
     → 实际到手年化会低于指数年化

  3. 申赎时间：QDII基金赎回到账较慢
     → 通常T+7到T+10个工作日
     → 流动性不如普通基金

  4. 汇兑成本：基金公司换汇有买卖价差
     → 每次申购赎回都有隐性汇兑成本
```

### 9.7 回测不等于未来

```
本报告中所有数据基于历史回测，重要提醒：

  1. 历史年化12%不代表未来也是12%
     → 可能在8-15%之间波动，也可能更低

  2. "每次都回来了"是历史事实，不是未来保证
     → 如果出现超长周期衰退（如日经225跌了34年），
       回本时间可能远超5年

  3. 回测未扣除实际费用
     → 基金管理费、托管费、汇兑损失
     → 实际到手收益会比回测低1-2个百分点

  4. 黑天鹅事件无法预测
     → 战争、疫情、政策突变等
     → 任何模型都无法覆盖所有极端情况

建议：
  → 用 compound_calculator.py 分别测算 8%、10%、12%、15% 不同场景
  → 做好最坏情况的心理准备
  → 建议只用"闲钱"投资（未来5-10年不需要的钱）
```

## 附录：常见问题

### Q: 纳指会不会像日经225一样失落30年？

```
日经225 vs 纳指100：

日经225失落的原因：
  1. 日本央行疯狂加息（贴现率从2.5%→6%）
  2. 银行业崩溃，整个金融系统瘫痪
  3. 人口老龄化，内需萎缩
  4. 企业治理差，交叉持股，缺乏创新

纳指100不具备这些条件：
  1. 美联储有完整的货币政策工具，08年后监管大幅加强
  2. 纳指100不含金融股，不受银行危机直接影响
  3. 美国人口结构健康（移民补充年轻劳动力）
  4. 纳指成分股是全球最具创新力的公司，持续创造增量价值
  5. 优胜劣汰机制确保指数始终反映最强企业

同时也要了解：
  → 这并不意味着纳指"不可能"出现长期低迷
  → 只是概率远低于日经225
  → 任何投资都有不确定性
```

### Q: 如果美国经济崩了怎么办？

```
如果美国经济崩到纳指归零的程度：
  → 全球经济都会崩
  → 你的股票、房子、现金、所有资产都会受影响
  → 那时候要担心的不是投资亏损，而是生存问题

所以这个担心在操作层面意义有限。
你能做的最好的准备就是：在正常世界里做合理的投资决策。
```

### Q: 为什么不等跌了再买？

```
见第二章 2.4 和 2.5 的回测数据。

核心：纳指70%以上的月份在上涨。
你等跌的时候，大概率在踏空。
回测表明"等跌再买"的年化低于固定定投。
```

### Q: 为什么不定投沪深300？

```
沪深300 vs 纳指100 的核心差异：

沪深300的成分股缺陷：
  → 沪深300 = A股市值最大的300家公司
  → 但中国最优秀的互联网/科技公司大多不在A股上市：
      腾讯、美团 → 港股
      阿里、拼多多、京东 → 美股/港股
      字节跳动、滴滴 → 未上市/海外
  → 沪深300的成分股以银行、地产、白酒、传统能源为主
  → 无法追踪到中国经济增长中科技行业的红利

创新高能力的差异（基于历史数据）：

  沪深300：
    2007年高点: 5891点
    2024年:     约3500点
    → 17年过去了，仍然没有超过2007年的高点
    → 期间多次"看似创新高"后又跌回来

  纳指100：
    2000年高点: 4800点
    2024年:     约18000点+
    → 24年后翻了近4倍
    → 每次回调后都能创出更高的新高

回测对比（建议用第八章工具自行验证）：

  指数          近10年年化    近20年年化    2007高点至今
  ────────────────────────────────────────────────────────
  纳指100       ~18%         ~14%         +约300%
  沪深300       ~5%          ~5%          -约40%（仍未回本）

  → 纳指的回撤后能创新高，沪深300的回撤后可能十几年都在横盘
  → 这不是偏见，是两个指数底层公司质量的差异

底层原因：
  → 纳指100的成分股（苹果/微软/英伟达）是全球赚钱能力最强的公司，
    利润持续增长 → 股价持续增长 → 指数持续创新高
  → 沪深300的成分股以传统行业为主，盈利增长乏力，
    银行/地产面临政策调控，白酒/消费受经济周期影响大
  → 纳指有优胜劣汰机制，沪深300也有，但优质替代者较少

结论：
  → 沪深300不是"不好"，只是从定投角度来看，长期向上的确定性远低于纳指
  → 定投需要"长期震荡向上"的资产才能发挥最大效果（见第二章 2.3）
  → 沪深300更像是"长期震荡横盘"，定投的微笑曲线难以充分展开
```

### Q: 为什么定投指数而不是个股？

```
定投指数 vs 定投个股的本质区别：

定投指数（如纳指100）：
  → 买入的是一篮子公司（100家），天然分散
  → 优胜劣汰机制自动换血：衰退的公司被踢出，新兴的公司被纳入
  → 即使其中几家公司的基本面恶化，对整体指数的影响有限
  → 指数的长期走势取决于经济大趋势，而非单一公司的命运
  → 对"长期向上"有较高的信心基础

定投个股（如某只股票）：
  → 买入的是单一公司，所有风险集中
  → 公司可能因为管理失误、技术迭代、政策变化、财务造假等原因衰退
  → 定投一个持续下跌的个股，越投越亏，没有底

历史上的例子：

  公司          定投10年结果           原因
  ────────────────────────────────────────────────────────
  诺基亚        亏损80%+              智能手机时代掉队，再也没回来
  柯达          几乎归零               数码相机替代胶卷，破产退市
  雅虎          大幅亏损               搜索/广告被谷歌碾压，最终被收购
  中国石油(A股)  亏损60%+              2007年上市即巅峰，17年未回本
  中国平安       长期横盘              传统金融增长乏力

  → 这些公司在各自的黄金时代都曾被认为"永远涨"
  → 但个股的衰退可能是永久性的，定投也救不回来

  对比纳指100：
  → 诺基亚衰退了 → 指数自动换成了苹果/三星
  → 雅虎衰退了 → 指数自动换成了谷歌/Meta
  → Intel掉队了 → 指数自动换成了英伟达/AMD
  → 指数的定投者不受单家公司衰退的影响

定投的底层逻辑要求：
  → 定投需要标的"长期震荡向上"
  → 需要对这个方向有持续的信心
  → 个股的"长期向上"依赖于单一公司的持续竞争力，不确定性很高
  → 指数的"长期向上"依赖于整个经济/产业趋势，确定性高得多

一句话对比：
  → 定投个股 = 把信心押在一家公司身上
  → 定投指数 = 把信心押在一个产业趋势上
  → 后者的容错率远高于前者
```

### 指数与基金类

| 术语 | 解释 |
|------|------|
| **纳斯达克100（Nasdaq-100 / NDX）** | 美国纳斯达克交易所中按市值排名前100的非金融公司组成的股票指数，以科技股为主 |
| **标普500（S&P 500）** | 美国最大的500家上市公司组成的指数，覆盖所有行业，是最常用的美股基准 |
| **纳指100联接基金** | 国内基金公司发行的、跟踪纳指100指数的公募基金（QDII类型），用人民币即可购买 |
| **QDII基金** | "合格境内机构投资者"基金，允许国内投资者通过基金公司投资海外市场（如美股） |
| **ETF** | 交易所交易基金（Exchange-Traded Fund），可以在股票市场像股票一样买卖的基金 |
| **联接基金** | 以投资某只ETF为目标的场外基金，适合没有股票账户的投资者通过银行/支付宝等渠道购买 |
| **成分股** | 组成某个指数的具体股票，如纳指100的成分股就是那100家公司 |
| **基金净值（NAV）** | 基金的总资产减去负债后除以总份额，即每份基金值多少钱 |

### 投资策略类

| 术语 | 解释 |
|------|------|
| **定投（DCA）** | 定期定额投资（Dollar-Cost Averaging），每隔固定时间投入固定金额，不择时 |
| **微笑曲线** | 定投过程中，先跌后涨的走势下，低位买入更多份额使平均成本降低，最终收益大于净值涨幅的现象 |
| **择时** | 试图判断市场高低点来决定买入或卖出的时机 |
| **踏空** | 因为等待更低价格而没有买入，结果价格上涨错过盈利机会 |
| **逃顶** | 试图在市场最高点卖出 |
| **抄底** | 试图在市场最低点买入 |
| **优胜劣汰机制** | 指数定期按市值调整成分股的规则，市值大的纳入、小的剔除，自动完成选股 |
| **长期持有** | 买入后不因短期波动卖出，持有多年甚至 decades，让复利发挥作用 |

### 收益与风险类

| 术语 | 解释 |
|------|------|
| **年化收益** | 把一段时间内的总收益折算成每年的平均收益率，方便不同时间段之间比较 |
| **总回报（Total Return）** | 包含股价涨跌和分红再投资的完整收益，比单纯看股价涨幅更准确 |
| **复利** | 利息产生利息的效应。今年的收益会在明年产生新的收益，随时间推移呈指数增长 |
| **回撤** | 从最高点到最低点的跌幅，比如从100跌到70就是-30%回撤 |
| **最大回撤** | 历史上最惨的一次下跌幅度，衡量投资风险的重要指标 |
| **年化波动率** | 收益率的波动程度，数值越大说明涨跌越剧烈 |
| **4%法则** | 每年从投资组合中提取4%用于生活开销，基于历史数据，30年内资产不会耗尽的概率很高 |

### 市场术语类

| 术语 | 解释 |
|------|------|
| **市值** | 公司在股票市场上的总价值 = 股价 × 总股数 |
| **权重** | 某只股票在指数中占的比重，纳指100按市值加权，市值越大权重越高 |
| **护城河** | 企业保护自己不被竞争对手抢走利润的壁垒（如品牌、技术专利、网络效应） |
| **基本面** | 企业的实际经营状况（营收、利润、现金流等），区别于短期股价波动 |
| **黑天鹅事件** | 极度罕见、难以预测但影响巨大的事件（如2020年疫情） |
| **泡沫** | 资产价格远超其内在价值的状态，通常伴随过度乐观和投机行为 |
| **熔断** | 股市暴跌到一定程度时自动暂停交易的机制，给市场冷静时间 |

### 国内投资特有

| 术语 | 解释 |
|------|------|
| **限购** | 基金公司限制每日申购金额（如每天最多买1000元），QDII基金因外汇额度紧张经常出现 |
| **申购/赎回** | 买入基金叫申购，卖出基金叫赎回 |
| **管理费** | 基金公司每年收取的管理报酬，从基金资产中直接扣除，QDII一般约1.0-1.2%/年 |
| **托管费** | 托管银行收取的费用，一般约0.2-0.25%/年 |
| **汇兑损失** | 人民币换美元、美元换人民币过程中的成本，买入和卖出都有价差 |
